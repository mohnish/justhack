Justhack

A sweet timer for your hack needs

Download this project as a .zip file Download this project as a tar.gz file

JustHack is a timer that can be used to set time for a specific task. If you have secondary monitors, then you can simply use JustHack on it. It is still under development and once I am done with the basic functionality, I will host it at http://justhack.in.

Stack: Node.js, Socket.io, ExpressJS
